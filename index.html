Running x86 software on Asahi Linux

Asahi Linux is a series of packages that allow Linux to boot and run on Apple Silicon. As AS is ARM-based, and a lot of closed-source software is x86-only, there needs to be a way to translate between the two architectures in order to achieve the broad compatibility enjoyed by x86-based PCs.
